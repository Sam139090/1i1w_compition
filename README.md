# **Structural Collapse Risk Prediction System**

This repository contains a full, production‑grade system designed to assist with structural safety assessment and earthquake preparedness. The project provides an end‑to‑end workflow: collecting building information, running a deep‑learning model to predict the probability of collapse, generating official PDF safety reports, simulating 2D earthquake behavior, verifying report authenticity, and supporting users through an AI‑powered assistant specialized in structural safety.

Although I initially assembled a team for this competition, none of the teammates contributed, and the entire project—design, modeling, backend, frontend, simulation, and integration—was fully developed by me from zero to completion.

---

## **Project Overview**

The system allows users to input 8 key building parameters:

- Shear Wall  
- Number of Floors  
- Building Age  
- Area  
- Expected Earthquake Magnitude  
- Soil Liquefaction Risk  
- Structure Type  
- Soil Type  

After the user submits the form, the data is processed by the backend and evaluated by a deep‑learning model. The output includes:

- Probability of structural collapse  
- Safety recommendations  
- Color‑coded risk level for easy interpretation  

Risk classification colors:

- **0–25%** → Green  
- **26–50%** → Yellow  
- **51–75%** → Orange  
- **76–100%** → Red  

These visual cues help non‑technical users understand the severity of risks instantly.

## **Key Features**

### **1. Deep Learning Model for Collapse Prediction**
I trained a custom LMP‑based neural network model specifically for structured/tabular data. Since an appropriate real‑world dataset wasn’t available, I created a synthetic dataset with realistic relationships and constraints.

Supporting components used:

- Feature engineering  
- Scaling  
- Model tuning  

The final model achieved **100% accuracy within a 1% tolerance**.  
Earlier, I experimented with an XGBoost model, which achieved 75% accuracy at 5% tolerance, but I switched to deep learning to reach the required performance.

---

### **2. Safety Recommendation Engine**
Based on conditional logic and building attributes, the system provides tailored safety recommendations. These guidelines help users understand which structural reinforcements or inspections might be necessary.

---

### **3. PDF Report Generator (With Anti‑Forgery System)**
Users can export their building analysis into a professionally designed PDF document.

The report includes:

- Input building parameters  
- Collapse probability  
- Safety recommendations  
- Unique **report_id**  
- Timestamp  
- Embedded **hologram** element to prevent forgery  

Each generated report is also stored in a local JSON file so it can be authenticated later. This mechanism allows engineers to rely on the report as an official, verifiable document.

---

### **4. PDF Verification System**
Users can input a **report_id** and the exact timestamp of the PDF.  
The backend searches the JSON registry and determines:

- Whether the PDF is valid  
- Whether it was ever generated by the system  
- The building information associated with it  

Invalid or altered documents are immediately flagged.

---

### **5. 2D Earthquake Simulation**
This module provides an offline, physics‑inspired 2D simulation where users can input building and earthquake parameters (including magnitude).  
Due to restrictions of the national network environment, the entire simulator was built using **vanilla JavaScript**, **HTML**, and **CSS**, without external libraries.

The simulation visualizes how a structure with the given properties would react under an earthquake of magnitude *n*, helping users understand vulnerabilities more intuitively.

---

### **6. ANKA – AI Safety Assistant**
The system includes a dedicated AI chatbot named **ANKA**, focused on structural safety and earthquake‑related guidance.

Features:

- Powered by **Claude 4.6 Sonnet API**  
- Has access to the user’s building parameters  
- Provides personalized advice  
- Includes **STT (speech‑to‑text)** using Whisper API  
- Fully integrated into the UI (AI.html)

## **Purpose**

The goal of this system is to provide a practical, accessible, and credible safety evaluation tool for buildings under earthquake conditions. It is designed for both general users and structural engineers, offering a blend of scientific analysis, predictive modeling, and actionable recommendations.
